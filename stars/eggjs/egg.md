Quickstart
----------

Follow the commands listed below.

$ mkdir showcase && cd showcase
$ npm init egg --type=simple # Optionally pnpm create egg --type=simple
$ npm install
$ npm run dev
$ open http://localhost:7001

> Node.js >= 14.20.0 required.
