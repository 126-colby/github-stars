wxpublic-python
===============

Wechat public platform send/response message with python

### Usage

It depends on this library:

-   BeautifulSoup installed as below:
    
    ```
    pip install beautifulsoup4
    ```
    
-   web.py installed as below:
    
    ```
    pip install web.py
    ```
    

run server as below:

```
python weixin.py  port
```
